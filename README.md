# 🌐 Portfolio Website using HTML & CSS

This project is a personal portfolio website built using HTML and CSS to showcase skills, projects, and personal information in a clean and professional manner. A portfolio website serves as an online resume, helping developers present their work, experience, and achievements effectively. :contentReference[oaicite:0]{index=0} The website includes multiple sections such as an introduction, about, skills, projects, and contact information, all designed with a responsive and modern layout.

---

## 🚀 Features

- 🧑‍💻 Personal introduction section  
- 📄 About me section  
- 🛠️ Skills showcase  
- 📁 Project portfolio display  
- 📞 Contact section  
- 🎨 Clean and responsive design

## 🛠️ Technologies Used

- HTML5  
- CSS3
